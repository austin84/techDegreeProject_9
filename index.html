HTML Forms

This is a responsive, "mobile-first" HTML form for registering to receive a newsletter. It required the use of a wide variety of HTML form input types and attributes. This form demonstrates an understanding of HTML forms, CSS, and Responsive design.

A Filterable Lightbox Gallery

This is an interactive photo gallery I made using JavaScript and jQuery. The thumbnails and photos are searchable and filter in real-time. When the user clicks on a thumbnail, the photo displays in a lightbox where they will find navigation and further info about the photo.

Wheel of Success

This is a browser based app similar to 'Wheel of Fortune', a word guessing game. Players click letters on an onscreen keyboard and try to guess a random phrase. Using Javascript, I created an array of phrases that could be chosen from randomly. The player gets to choose until they solve the phrase or make five(5) incorrect guesses. Give it a play.

Responsive Graphs, Real-Time Data

This is a web application dashboard. It acts like a control panel for interacting with a web app. This dashboard is complete with JavaScript-driven charts and graphs. You can manipulate the graphs in real-time with static data created ahead of time. Easily connects to your favorite API.

Connect Four

This is a remake of the old classic, Connect Four! This web app was made using Object Oriented JavaScript. Two(2) Players can compete against eachother using just a keyboard. This game demonstrates a working knowledge of Classes and Object manipulation in JavaScript.

An API Sourced Employee Database

This site uses the Random User Generator API (https://randomuser.me/) to pull in data on 12 random "Employees". It then uses that data to build a employee directory. The employee data supplies both the employee cards and a modal that pops-up. This project demonstrates a knowledge of how to retrieve data utilizing the Fetch API, as well as how to handle that data when returned in the form of JSON.
